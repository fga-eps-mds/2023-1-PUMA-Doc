    Estrutura Analítica de Projetos

    1. Introdução

    Segundo Vargas (2015, p. 51), a Estrutura Analítica do Projeto (EAP), também conhecida como WBS - Work Breakdown Structure, é a ferramenta de gerenciamento do escopo do projeto. Cada nível descendente do projeto representa um aumento no nível de detalhamento do projeto, como se fosse um organograma (hierárquico). O detalhamento pode ser realizado até o nível desejado, apresentando dados genéricos ou detalhados. [1]

    Uma EAP geralmente é apresentada como um diagrama de árvore, com o nível mais alto da hierarquia representando o objetivo geral do projeto. Os níveis abaixo do nível mais alto representam as entregas intermediárias necessárias para alcançar o objetivo geral, e os níveis mais baixos representam as atividades e tarefas específicas necessárias para produzir cada entrega. [2]
